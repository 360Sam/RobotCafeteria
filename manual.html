Robot Manual

The following is a summary of the robot's programming language. The robot understands most algebra and arithmetic, including the use of letters to stand in for numbers. But the letter must first be given a meaning before it can be used.

Standard Operations
For the examples, let \(A=10\)
Operation Symbol Examples Description
Addition + 4+37
A+313		 Addition is written with the plus sign.
Subtraction - 4-31
4-A -6		 Subtraction is written with the minus sign.
Multiplication * 4*312
4*A40		 Multiplication is written with the asterix (or 'star') sign. The robot does not understand multiplication any other way. It will not work to write \(4\times3\), \(4(3)\) or \(4A\).
Division / 4/31.333...
A/42.5		 Division is done with the forward slash. It will not work to use a backslash, \.
Grouping (Association) ( ... ) (4-3)/A0.1
4-(3/A)3.7		 Grouping is done with parentheses. The robot does not understand other grouping symbols like [ ... ].
Special Functions
For the examples, let \(A=10\), \(B=-4\), \(C=1.5\)
Function Mathematical Notation Robot Language Symbol Examples Description
Absolute Value \(\lvert n \rvert\) absolute(n) absolute(-4)4
absolute(C)1.5		 The absolute value function makes negative numbers positive, and leaves positive numbers unchanged.
Maximum \(max(x,y,z, ...)\) max(x,y,z, ...) max(1,2)2
max(A,C,-3,0)10		 The maximum function chooses the greatest number from a list. Numbers in the list must be separated by commas. The list can have any length as long as there is at least one thing.
Minimum \(min(x,y,z, ...)\) min(x,y,z, ...) min(1,2)1
max(A,C,-3,0)-3		 The minimum function chooses the least number from a list. Numbers in the list must be separated by commas. The list can have any length as long as there is at least one thing.
Exponents (powers) \(x^n\) power(x,n) power(3,2)9
power(B,2)16		 The exponent function multiplies a number by itself a certain number of times.
Square Root \(\sqrt{x}\) root(x) root(4)2
root(100)10		 The square root function \(\sqrt{x}\) is the inverse of the second power function \(x^2\).
Round \([x]\) round(x) round(1.4)1
round(3.592389)4
round(-2.6)-3		 The round function rounds any number to the nearest integer using the usual rules.
Round Down \(\lfloor x\rfloor\) rounddown(x) rounddown(1.4)1
rounddown(3.592389)3
rounddown(-2.1)-3		 The round down function rounds every number that is not an integer to the next lowest integer. If round down is used on an integer, it returns the integer unchanged.
Round Up \(\lceil x\rceil\) roundup(x) roundup(1.4)2
roundup(3.592389)4
roundup(-2.6)-2		 The round up function rounds every number that is not an integer to the next greatest integer. If round up is used on an integer, it returns the integer unchanged.
